# x-powered-by の無効化

Next.js はデフォルトで `x-powered-by` ヘッダーを追加します。これをオプトアウトするには、 `next.config.js` を開いて `poweredByHeader` という設定を無効化してください:

```js
module.exports = {
  poweredByHeader: false
};
```
