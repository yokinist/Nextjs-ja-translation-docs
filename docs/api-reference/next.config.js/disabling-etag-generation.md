# ETag 生成の無効化

Next.js はデフォルトですべてのページの [ETag](https://ja.wikipedia.org/wiki/HTTP_ETag) を生成します。キャッシュ戦略によっては、HTML ページの ETag 生成を無効化した方が良いでしょう。

`next.config.js` の `generateEtags` オプションを無効化します:

```js
module.exports = {
  generateEtags: false
};
```
